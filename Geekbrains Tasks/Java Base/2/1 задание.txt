var a = 1, b = 1, c, d;
c = ++a; alert(c);           // 2
d = b++; alert(d);           // 1
c = (2+ ++a); alert(c);      // 5
d = (2+ b++); alert(d);      // 4
alert(a);                    // 3
alert(b);                     //3

На второй строке написано,что с = а+1,1 +1 =2,значение переменной a теперь также становится 2;
На третьей строке в выражении: d=b+1,но переменной d возвращается старое значение,равное единице,переменная b теперь равняется 2
На четвертой строке мы к числу 2 прибавляем значение a,равное двум,к которому прибавляется ещё единица, таким образом,мы к 2 прибавляем 3,получаем 5
Наконец,на пятой строке,мы к числа 2 прибавляем старое значение b,которое в прошлый раз стало равно 2,2+2 =4,теперь b получает новое значение:2+1=3
Таким образом,переменная а и b теперь равны 3
Здесь показана работа постфиксной и префиксной формы инкремента
